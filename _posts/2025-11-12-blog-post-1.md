## LLama Factory 安装
***
通过`nvcc -v `检查cuda版本，然后执行以下代码
```bash
git clone --depth 1 https://github.com/hiyouga/LLaMA-Factory.git
cd LLaMA-Factory
pip install -e ".[torch,metrics]"
```
通过`llamafactory-cli version`检查是否安装成功
